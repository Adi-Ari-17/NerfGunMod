# NerfGunMod
A nerf gun mod that (hopefully) will help make my gun shoot faster. 

# Parts

  - Nerf Gun (I used the Dart Zone BF-2) (https://www.target.com/p/dart-zone-blitzfire-x2-combo-2-pack-quickshot-dart-blasters/-/A-52253560)
  - Wires (I mostly used male to male) Various
  - Battery Pack (I used 4 AA batteries serrially connected to have a higher current then a my 9v) Various
  - 2 DC motors (Should be fast) Various
  - Access to a 3d printer (For our mount) Various
  - Soldering Set (Tape works but is hard to use) Various
  - Battery Pack
