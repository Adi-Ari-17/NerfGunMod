# NerfGunMod
A nerf gun mod that (hopefully) will help make my gun shoot faster. 

# Parts

  - Nerf Gun (I used the Dart Zone BF-2)
  - Wires (I mostly used male to male)
  - Battery Pack (I used 4 AA batteries serrially connected to have a higher current then a my 9v)
  - 2 DC motors (Should be fast)
  - Access to a 3d printer (For our mount)
  - Soldering Set (Tape works but is hard to use)
